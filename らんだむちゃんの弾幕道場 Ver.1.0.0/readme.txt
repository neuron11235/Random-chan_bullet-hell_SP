●操作方法
<メニュー・ステージセレクト画面>
矢印キー：選択
Zキー：決定
Xキー：戻る
左Shiftキー + 矢印キー：ウィンドウ位置の調整(メニュー画面でのみ操作可能)

<ゲーム中>
矢印キー：移動(左Shiftキーを押しながらで低速移動)
Zキー：ショット
Escキー：インスタントデス(誤ってステージ選択した時など用)

●対象OS：Windowsのみ

●このゲームについて
スペルプラクティス形式の弾幕STGです。
難易度ごとにレベル分けされていて、その中にいくつかのステージがあります。
ガチ避け力を鍛え、より高レベルの弾幕の攻略を目指せ!
